# Music Player

## Installation

Install for Linux (POSIX).
```bash
$ git clone https://github.com/alipasaturhan/Music-Player
$ cd Music-Player
$ wine MusicPlayer.exe
```
Install for Windows (NT).
```bash
$ Download as ZIP
$ Extract files and run MusicPlayer.exe
```
## Environment
For Linux
```bash
-> wine i386
```
For Windows
```bash
-> .NET 4.7.2
```

Supported audio files: .mp3
